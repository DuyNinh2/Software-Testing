# Project Overview:
Our team perform software testing on a Java-based BankAccount class and conducted automated UI testing for eBay using Selenium. The focus was to apply both black-box and white-box testing methodologies in real-world test scenarios and learn hands-on test design, execution, and reporting.

# BankAccount Class Testing
- Performed unit testing using JUnit for various methods in the BankAccount class: constructor, getters/setters, deposit, withdraw, and transfer.
- Applied Black-box testing to validate input validation, exception handling, and system behavior without accessing internal logic.
- Used White-box testing to ensure full code coverage, including statements, paths and conditions.
- Designed and implemented parameterized tests to systematically verify multiple input combinations across test cases.
- Exported test case results to CSV file for reporting and evaluation.
- Achieved high test coverage by organizing test classes into suites for modular testing.

# Selenium Automation – eBay Web UI
- Automated functional test cases for eBay’s web UI using Selenium WebDriver.
- Implemented test scripts to simulate real user actions (typing, clicking, asserting element visibility/text).
- Used XPath/CSS Selectors to interact with dynamic page elements.
